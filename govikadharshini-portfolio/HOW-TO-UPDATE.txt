ADD A PROJECT (only edit js/projects-config.js)
================================================
Open js/projects-config.js and change:

  const PROJECTS = [];

to something like:

  const PROJECTS = [
    {
      title: "HealthAI",
      description: "Disease prediction using Python and ML.",
      tags: ["Python", "AI"],
      image: "images/healthai.jpg",
      link: "https://github.com/govika-ef/your-repo"
    }
  ];

- link empty ""  → card shows "Link coming soon" (no click)
- link with URL  → opens in NEW TAB when user clicks
- Save file and refresh browser — no other file changes needed
